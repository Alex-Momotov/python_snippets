	PYCHARM
1. Do "pipenv install" in the desired project sub-directory.
2. Open Project Interpreter menu and add New Virtual Environment.
3. Specify path to Python interpreter where pipenv created it: /Users/omomotov/.local/share/virtualenvs/MyEnvironment/bin/python

------------------------------------------------------------------------------------------
pipenv install

Create a virtual environment associated with current directory, if one doens't already exist. Automatically generates Pipfile and Pipfile.lock files in current directory. 
Environment itself is created in /Users/omomotov/.local/share/virtualenvs with the name of the current folder. Also used to create environment from existing Pipfile files.
------------------------------------------------------------------------------------------
pipenv shell

Activate environment associated with current directory. If directory does not contain Pipfile and Pipfile.lock, new environment will be created in current directory in same way as "pipenv install".
------------------------------------------------------------------------------------------
pipenv run <command>

Run a command from environment in current folde, without explicitly activating it first.
------------------------------------------------------------------------------------------
exit

Exit current environment shell and deactivate the environment.
------------------------------------------------------------------------------------------
pipenv --venv

Display directory where virtual environment associated with current directory is located.
------------------------------------------------------------------------------------------
pipenv --rm

Remove environment assiciated with current directory from /Users/omomotov/.local/share/virtualenvs. Need to make sure the directory completely disappeared.
